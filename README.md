# SaaS Analytics Dashboard

A professional dashboard that connects to Stripe accounts to visualize MRR, Churn Rate, LTV, and other critical SaaS metrics.

## Features
- **Key Metrics:** Real-time tracking of Monthly Recurring Revenue (MRR), Active Users, Churn, and Lifetime Value (LTV).
- **Revenue Growth:** Interactive area charts to visualize revenue trends over time.
- **User Acquisition:** Bar charts to monitor user growth and retention.
- **Enterprise Ready:** Clean, professional UI with real-time feedback and high-performance components.

## Setup

### App (Next.js)
1. Go to `root`
2. Run `npm install`
3. Run `npm run dev`

Note: This project is a frontend-only demonstration with mock data integration in `lib/stripe-mock.ts`.

## Tech Stack
- **Frontend:** Next.js, Tailwind CSS, Lucide React, Recharts
- **Data Integration:** Mock API integration for high-performance UI demonstration.
