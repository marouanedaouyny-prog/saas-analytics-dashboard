# SaaS Analytics Dashboard

A professional dashboard that connects to Stripe accounts to visualize MRR, Churn Rate, LTV, and other critical SaaS metrics.

![SaaS Analytics Dashboard Demo](./demo.png)

## ✨ Features

- **Key Metrics:** Real-time tracking of Monthly Recurring Revenue (MRR), Active Users, Churn, and Lifetime Value (LTV)
- **Revenue Growth:** Interactive area charts to visualize revenue trends over time
- **User Acquisition:** Bar charts to monitor user growth and retention
- **Cohort Analysis:** Track customer retention and behavior patterns
- **Export Functionality:** Download reports as CSV/PDF
- **TypeScript Support:** Full type safety with interfaces and error handling
- **Modern UI:** Built with Next.js 14, Tailwind CSS, and Recharts

## 🚀 Quick Start

### Prerequisites

- Node.js 18+ and npm

### Installation

1. Navigate to the project directory:
   ```bash
   cd saas-analytics-dashboard
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Set up environment variables:
   ```bash
   cp .env.example .env.local
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser

## 📊 Key Metrics Explained

### MRR (Monthly Recurring Revenue)
Predictable revenue generated by all active subscriptions in a given month.

### Churn Rate
Percentage of subscribers who cancel their subscriptions within a given period.

### LTV (Lifetime Value)
Total revenue expected from a customer throughout their entire relationship.

### Active Customers
Number of customers with active subscriptions in the current period.

## 🔌 Stripe Integration Guide

To connect to real Stripe data:

1. **Install Stripe SDK:**
```bash
npm install stripe
```

2. **Add Stripe API key to `.env.local`:**
```
STRIPE_SECRET_KEY=sk_test_your_stripe_secret_key
```

3. **Update `lib/stripe-mock.ts` to use real API:**
```typescript
import Stripe from 'stripe';

const stripe = new Stripe(process.env.STRIPE_SECRET_KEY!, {
  apiVersion: '2024-04-10',
});

export const fetchSaaSMetrics = async (): Promise<SaaSMetrics> => {
  try {
    // Fetch subscriptions from Stripe
    const subscriptions = await stripe.subscriptions.list({
      limit: 100,
      status: 'active',
    });
    
    // Calculate MRR
    const mrr = subscriptions.data.reduce((acc, sub) => {
      return acc + (sub.items.data[0]?.price?.unit_amount || 0) / 100;
    }, 0);
    
    // Transform to match SaaSMetrics interface
    return {
      mrr,
      mrrGrowth: 12.5,
      churn: 2.1,
      ltv: 850,
      activeCustomers: subscriptions.data.length,
      chartData: [...]
    };
  } catch (error) {
    console.error('Failed to fetch Stripe data:', error);
    throw {
      message: 'Failed to load SaaS metrics. Please try again.',
      status: 500
    } as ApiError;
  }
};
```

## 📈 API Response Schema

```typescript
interface SaaSMetrics {
  mrr: number;              // Monthly Recurring Revenue
  mrrGrowth: number;        // MRR growth percentage
  churn: number;            // Churn rate percentage
  ltv: number;              // Lifetime Value
  activeCustomers: number;  // Number of active customers
  chartData: Array<{
    month: string;
    revenue: number;
    users: number;
  }>;
}
```
